## üìù **Field Guidelines**

### Full Name
- **Format**: First Name + Last Name
- **Examples**: 
  - `John Smith`
  - `Sarah Johnson`
  - `James Mwanza`
  - `Grace Banda`
- **Tips**: Use your real name for professional identification

### Username
- **Requirements**: Minimum 3 characters
- **Format**: Lowercase letters, numbers, underscores
- **Examples**:
  - `jsmith_admin` (first initial + last name + role)
  - `sarah_env` (first name + department)
  - `mwanza123` (last name + numbers)
  - `resident_grace` (role + first name)
- **Tips**: Make it memorable but professional

### Password
- **Requirements**: Minimum 6 characters
- **Recommendations**: 8+ characters with mix of letters, numbers, symbols
- **Good Examples**:
  - `TSF2024!Safe` (project + year + symbol + word)
  - `EnviroSafe123` (department + word + numbers)
  - `Community2024` (role + year)
  - `SafetyFirst456` (phrase + numbers)
- **Tips**: Use something related to TSF safety or your role

## üîí **Security Tips**

1. **Use Strong Passwords**: Mix letters, numbers, and symbols
2. **Keep Credentials Safe**: Don't share your login details
3. **Professional Usernames**: Use work-appropriate usernames
4. **Real Names**: Use your actual name for accountability
5. **Unique Passwords**: Don't reuse passwords from other systems

---

## üìû **Need Help?**

If you have trouble creating your account:
1. Check that all fields are filled correctly
2. Ensure passwords match exactly
3. Make sure username is at least 3 characters
4. Verify password is at least 6 characters
5. Try a different username if "already exists" error appears
